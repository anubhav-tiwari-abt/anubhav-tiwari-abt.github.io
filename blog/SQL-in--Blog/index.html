All you need to know

If you're in the field of data, no matter what your role is, whether it is a data scientist, data engineer, machine learning engineer etc., you'll have to, at some point, deal with tabular data or structured data. Big data is mostly stored in some form of database, and for ease of operation on the dataset, you need the right tools, and the main tool that you'll always need, is SQL.

SQL or Structured Query Language, is an essential part of the data science toolkit, and it is essential to know how to write at least some basic queries. Here I'll try to go through most of the concepts with some SQL code and some sample problems that will cover all the SQL you need to know to write basic and some advanced queries. Expertise only comes with practice and day-to-day interaction with SQL will only improve the skills.

SQL Engines

SQL comes in several, flavors if you will. Each adding some features on top of the SQL core. Some examples of common and popular engines are :
  1. MySQL
  2. Microsoft SQL Server
  3. Postgre SQL
  4. Oracle SQL Engine
  5. IBM SQL Engine
Here I'll be using MySQL for all the examples and will be explaining concepts with MySQL in mind.
